title: Latency Measurement Tool
description: A specialized latency meter built for PlayStation's cloud gaming platform, used to benchmark local versus cloud input lag. An embedded spectrometer and <strong>STM32-based microcontroller</strong> integrate with DualSense controllers to simulate precise inputs and measure the display's response. I extended the firmware to make measurement game-agnostic, and added luminosity and color-shift detection- working in <strong>CIE Lab* color space</strong> with on-device calibration, so latency can be measured accurately during real gameplay rather than in a controlled test pattern.
