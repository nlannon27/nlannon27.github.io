title: SCP-173 (Garry's Mod)
description: A published Garry's Mod addon with over <strong>60,000 users</strong> and a <strong>5-star rating across 1,500+ reviews</strong>. SCP-173 can only move while no player is looking at it, so the main challenge was tracking line-of-sight for every player without overloading the server. Instead of the server checking visibility for everyone, each client runs its own lightweight check and reports when its view changes, which the server then verifies. I built it on a base of my own where adding a new SCP is as simple as dropping a file into a folder, the base scans it in and handles everything, including custom config options, so other developers can build and add their own SCPs with very little setup.
